# Western Tech Hub

A multi-page business website created as part of Western Governors University's D197 Version Control course. This project demonstrates the use of Git for version control while building and managing a static website with HTML, CSS, and JavaScript.

## Overview

Western Tech Hub is a sample technology company website that includes multiple pages for company information, services, customers, team members, and contact information. The project emphasizes proper project organization and version control practices.

## Features

- Multi-page website
- Responsive navigation menu
- Home, About, Services, Customers, Team, Vision, and Contact pages
- Custom CSS styling
- JavaScript for interactive functionality
- Organized project structure with separate HTML, CSS, JavaScript, and image files

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
